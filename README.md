# gogen
gocli终端应用

``` text
go run main.go -ui ，开启一个带简单ui的终端界面
go run main.go help  执行help命令
显示：
主程序:gogen_core 版本:v0.0.1
用法:Command [args...] [-Flag [flagArgs...] ...],可用指令:
  command   指令使用帮助信息
  genplugin 生成一个插件模板文档 eg: genplugin plugin/demo/plugin.go -nm demo_plugin -ver
  help      使用方法,简述
  quit      退出程序
  show
  ws        显示当前工作空间状态相关信息

```
